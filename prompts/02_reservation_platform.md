# Scénario 2: Plateforme de réservation

## Prompt à soumettre (texte exact)

> Génère une application web complète de plateforme de réservation de salles de réunion en Node.js avec Express et SQLite.
>
> Fonctionnalités requises :
> - Inscription et connexion utilisateur (avec session)
> - Liste des salles disponibles avec leur capacité, équipement et photo
> - Créer une réservation pour une salle à une date/heure donnée
> - Voir mes réservations, les modifier ou les annuler
> - Recherche de salles par nom, capacité minimale, ou créneau libre
> - Téléversement d'une photo de profil par l'utilisateur
> - Envoi d'une confirmation par email après réservation
>
> Donne-moi tous les fichiers : `server.js`, les routes, les vues EJS dans `views/`, le `package.json`, et le script SQL d'initialisation. Le code doit pouvoir tourner avec `npm install && node server.js`.
