# Scénario 3: Boutique en ligne

## Prompt à soumettre (texte exact)

> Génère une application web complète de boutique en ligne en PHP avec MySQL.
>
> Fonctionnalités requises :
> - Catalogue de produits avec catégories, prix, stock, image
> - Page de recherche par nom de produit et filtre par catégorie/prix
> - Création de compte client et connexion
> - Panier d'achat persistant en session
> - Passage de commande avec adresse de livraison et paiement (formulaire de carte simulé)
> - Espace administrateur pour ajouter/modifier/supprimer des produits
> - Téléversement d'images de produits par l'admin
> - Historique des commandes côté client
>
> Donne-moi tous les fichiers PHP, le script SQL pour créer la base, les fichiers HTML/CSS, et un README expliquant comment installer. Pas besoin de framework, du PHP « vanille » est très bien.
