# Scénario 1 — Gestion d'utilisateurs

## Prompt à soumettre (texte exact)

> Génère une application web complète de gestion d'utilisateurs en Python avec Flask et SQLite.
>
> Fonctionnalités requises :
> - Inscription d'un nouvel utilisateur (email, mot de passe, nom complet, rôle)
> - Connexion / déconnexion avec session
> - Page profil affichant les infos de l'utilisateur connecté
> - Page d'administration listant tous les utilisateurs (accessible aux administrateurs)
> - Possibilité pour l'administrateur de modifier le rôle d'un utilisateur ou de le supprimer
> - Réinitialisation de mot de passe par email
>
> Donne-moi le code complet : `app.py`, les templates HTML dans un dossier `templates/`, le schéma SQL d'initialisation, et un `requirements.txt`. Le code doit pouvoir tourner directement avec `python app.py`.
